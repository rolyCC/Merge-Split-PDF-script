# Merge-Split-PDF-script
PDF merge &amp; split
Herramienta de Fusión de PDFs (Merge PDF Tool)
Descripción en Español:
La herramienta de Fusión de PDFs permite combinar múltiples archivos PDF en un solo documento. Es ideal para consolidar documentos relacionados, como informes, capítulos de libros o artículos, en un único archivo para facilitar su manejo, distribución o archivo.

Uso:
Seleccione los archivos PDF que desea fusionar, haga clic derecho y seleccione "Enviar a", seguido de la herramienta de Fusión de PDFs. El script generará un nuevo documento PDF combinado, guardándose automáticamente en el mismo directorio de los archivos originales.

Description in English:
The Merge PDF Tool allows you to combine multiple PDF files into a single document. It's perfect for consolidating related documents such as reports, book chapters, or articles into one file for easier handling, distribution, or archiving.

Herramienta de División de PDFs (Split PDF Tool)
Descripción en Español:
La herramienta de División de PDFs está diseñada para separar un archivo PDF en varios documentos, cada uno con una sola página del original. Ideal para extraer páginas específicas de un documento mayor para su manejo, compartición o archivo individual.

Uso:
Seleccione el PDF que desea dividir, haga clic derecho y elija "Enviar a", seguido de la herramienta de División de PDFs. El script dividirá el PDF en páginas individuales, creando un archivo separado para cada una y ubicándolos en la misma carpeta del original.

Desarrollado por:
ChatGPT de OpenAI

Description in English:
The Split PDF Tool is designed to separate a PDF file into multiple documents, each containing a single page from the original. It's ideal for extracting specific pages from a larger document for individual handling, sharing, or archiving.

Usage:
Select the PDF you wish to split, right-click and choose "Send to", followed by the Split PDF Tool. The script will split the PDF into individual pages, creating a separate file for each, and place them in the same folder as the original.
